# Happy Birthday Suhani (23) - Website PRD

## Original Problem Statement
Build a FUNNY Gen-Z "Happy Birthday Suhani (23)" website as a single-page scroll experience with brown aesthetic theme.

## User Personas
- **Primary**: Suhani - Birthday girl turning 23 (March 2, 2003)
- **Secondary**: Friends viewing/sharing the site

## Core Requirements (Static)
- Brown aesthetic (her favorite color)
- Single-page scroll experience
- Mobile responsive
- Photo gallery with uploaded images
- Interactive elements (confetti, modals, flip cards)
- Inside joke references (Among Us, Monodeal, Saje, Four Olives)

## What's Been Implemented (March 2, 2026)

### Version 2 (Revised)
- **BLACK background** with brown/gold accents
- **Clean photo gallery** - 14 photos, NO captions
- **Removed cringe words**: slay, iconic, main character, legend
- **Sage corrected**: Shown as "Saje Sleep Well - Her 'vape'" (product, not aesthetic)
- **Structured sections**: Hero, Pictures, Quirks, Birthday, Among Us, Monodeal, Four Olives, Master's

5. Emergency Meeting (Among Us) section with voting
